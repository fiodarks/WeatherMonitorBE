# WeatherMonitor
## How to start app
Clone the repository:

```bash
git clone https://github.com/fiodarks/WeatherMonitorBE.git
cd WeatherMonitorBE
```
Run `docker-compose up --build`

## How to reach app
### Swagger
#### Local
http://localhost:8080/docs
#### DEV
https://limited-joleen-fiodarks-5b0af2c2.koyeb.app/docs

## Technical notes
App deployed on Koyeb service. As it is free it have problem while 
first start after a while it require extra time to restart.

DB is Postgres deployed on render as it is also free database.
